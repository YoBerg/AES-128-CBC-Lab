# Reference Solutions

These files contain one possible implementation for the lab exercises:

- `aes_cbc_lab_solution.py`: PKCS#7 padding, XOR, CBC encryption, and CBC decryption.
- `cbc_attack_lab_solution.py`: IV bit flipping and padding oracle block recovery.

Try solving the main lab files first. Use these only as a reference when you are stuck or when you want to compare approaches after your tests pass.
